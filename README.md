# Estruturas Elétricas Neoenergia Elektro

Aplicação Streamlit para consulta de estruturas elétricas com base nas normas DIS-NOR-013, DIS-NOR-014 e DIS-NOR-018.

## Arquivos necessários
- app.py
- app_estruturas_v2 (2).py
- requirements.txt
- normatives_text.json
- PDFs das normas na pasta do projeto

## Executar localmente
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy no Streamlit Cloud
1. Faça o push do repositório para o GitHub.
2. No Streamlit Cloud, selecione o repositório.
3. Defina o arquivo principal como app.py.
4. A aplicação será iniciada automaticamente.
